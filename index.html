✏️ How to open the Sweep Renamer
The Sweep Renamer is a local HTML file. Follow these steps to get it running on your computer.
1
Download the file
Save PIM - Sweep Renamer.html to your computer.
2
Locate the file
Open your Downloads folder (or wherever you saved it).
3
Open in Chrome
Right-click the file → Open with → Google Chrome.
4
If Chrome shows a warning
Click Keep if prompted. If blocked, go to chrome://settings/content/javascript and ensure JavaScript is allowed.
5
You're ready!
Drag and drop your .dat, .vna, or .pim files into the tool.
